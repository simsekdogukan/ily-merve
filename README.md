
# I Love You Merve ❤️

A beautiful, animated web page to express your love.

## Features
- **Particle Heart Animation**: A dynamic heart shape made of particles.
- **Text Reveal**: Smooth animation for the "I Love You Merve" message.
- **Interactive**: Particles react to mouse movement.
- **Responsive**: Works on both desktop and mobile devices.

## How to Deploy on Vercel

1.  **Push to GitHub**:
    Ensure your latest changes are pushed to your GitHub repository:
    ```bash
    git add .
    git commit -m "feat: add particle heart animation"
    git push origin main
    ```

2.  **Login to Vercel**:
    Go to [vercel.com](https://vercel.com) and log in with your GitHub account.

3.  **Add New Project**:
    - Click **"Add New..."** -> **"Project"**.
    - Import the `ily-merve` repository.

4.  **Configure**:
    - Framework Preset: **Other** (since it's plain HTML/CSS/JS).
    - Root Directory: `./` (default).
    - Click **Deploy**.

5.  **Done!**:
    Vercel will give you a live URL (e.g., `ily-merve.vercel.app`) to share with Merve.

## Customization
- To change the text, edit `index.html`.
- To change colors, edit `style.css` (look for `heartColors` in `script.js` too).
